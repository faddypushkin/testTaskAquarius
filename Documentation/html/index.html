Тестовое задание на вакансию стажёра програмиста микроконтроллеров (embedded-разработчика) в компанию «Аквариус»

Задача

Сторона A хочет передать стороне Б код от банковской ячейки, состоящий из 6 цифр (например, 748253), используя для этого голубиную почту. Никто не знает о договоренности А и Б пользоваться голубиной почтой для пересылки кода от ячейки, однако:

  • данный способ не позволяет передавать сообщения длиннее 5 цифр;
  • часть голубей может не долететь до получателя (их могут съесть по пути ястребы), но при благоприятных условиях голуби преодолевают расстояние между А и Б за время не более 1 часа;
  • покупать и обучать голубей очень дорого, поэтому желательно ограничить их число при пересылке кода.

Требуется реализовать ПО (клиент и сервер), демонстрирующее решение данной задачи с учетом следующих требований:

  • язык реализации: C;
  • компилятор: GCC;
  • микроконтроллер: STM32L433VC;
  • в качестве системы сборки проектов использовать GNU Make или CMake;
  • в качестве системы документирования кода проектов использовать Doxygen;
  • обмен данными между клиентом и сервером реализовать через UART;
  • допускается использование сторонних библиотек (находящихся в открытом доступе), не решающих за Вас задачу полностью, при указании ссылки на данную библиотеку и её документацию;
  • к решению задачи приложить описание реализованного алгоритма. Достаточно в виде текстового файла с разметкой Markdown;
  • исходные тексты разместить на сервисе GitFlic или GitHub в виде публичных репозиториев.

Использовалось:

  • язык компиляции: C;
  • компилятор GCC;
  • микроконтроллер STM32L433VCI3;
  • программа для предварительной настройки МК и инициализации начального кода STM32CubeMX;
  • библиотека HAL для STM32L4 (ссылка, документация);
  • система сборки проектов GNU Make;
  • ПО для программирования и отладки OpenOCD (команда make flash);
  • Система документирования Doxygen.
